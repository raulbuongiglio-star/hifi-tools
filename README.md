# HiFi Solution - Strumenti

Strumenti web per H&S Home Solution SRLS.

## Contenuto

- **index.html** - Generatore preventivi con calcolo automatico IVA

## Installazione su iPhone/iPad

1. Apri il sito in Safari
2. Tocca l'icona condividi (↑)
3. Seleziona "Aggiungi alla schermata Home"
4. L'app apparirà come icona sulla Home

## Deploy

Questo sito è hostato su Netlify e si aggiorna automaticamente ad ogni push su GitHub.
